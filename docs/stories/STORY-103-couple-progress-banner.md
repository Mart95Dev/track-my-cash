**Sprint :** v14 — Couple-First Onboarding  
**Priorité :** P2  
**Complexité :** S (2 points)  
**Epic :** couple-activation

---

## Objectif

Afficher une bannière de progression sur le dashboard pour les utilisateurs ayant choisi "En couple" mais sans partenaire connecté.

---

## Acceptance Criteria

- **AC-1** : Bannière "Invitez votre partenaire" si `onboarding_choice='couple'` et pas de couple actif
- **AC-2** : Bannière affiche le code d'invitation et bouton "Partager"
- **AC-3** : Bannière disparaît une fois le partenaire connecté
